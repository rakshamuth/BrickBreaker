I made breakout, featuring cats!

From the intro screen, press start to begin game play.
Any time during game play or game over screen, press select to go to intro screen.

In game screen, use left and right controls to move the paddle to bounce the ball off of it. Clear bricks by hitting them with ball.

Game is won when all bricks are cleared.

Bonus feature: this game features a difficulty curve, so getting the last few bricks isn't as easy as the first few!
